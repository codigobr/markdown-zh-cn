扩展语法

基于Markdown基础的高级功能。

  1. Markdown
  3. 扩展语法

介绍

约翰·格鲁伯（John Gruber）发布的Markdown的原始语法添加了一些用于编辑文档的基本元素，但一些用户不满意。

扩展语法试图通过通过新标记添加新元素来解决这种不满。

表，代码块，自动links生成，甚至脚注都是扩展基本Markdown语法的示例。

可以通过为Markdown处理器安装plugin或什至使用语言语法的较小变化来激活这些元素。

注意：您的Markdown应用程序很可能已经支持本文中介绍的元素，因此值得在学习本教程的同时对其中的每个元素进行测试。

Markdown口味

Markdown具有几种不同的风格，即基本语言语法的变体。

这些变体除了支持基本语法外，还添加了丰富文档编辑的元素。

与Markdown兼容的应用程序经常使用以下所示的一种语言：

CommonMark

GitHub-flavored Markdown (GFM)

Markdown Extra

R Markdown

MultiMarkdown

自动在Markdown中生成表

Markdown在这里使文档编辑更加容易，并且创建表格可能会变得更加乏味。

竖线和破折号的使用可能会使某些用户不满意，因此有趣的是，我们能够自动化Markdown中的表格格式化过程。

步骤1-单击此处打开我们的Markdown编辑器。

步骤2-单击自动表格生成图标以打开选项面板。

第3步-在选项面板中：

  1. 输入表格中的行数。
  2. 设置表中的列数。
  3. 在列中选择文本的对齐位置。
  4. 单击以确认并生成表结构。

第4步-填写每个单元格的值。

要创建被Markdown包围的代码块，请在内容之前放置三个引号`"，在内容之后放置三个引号```。

脚注

Markdown上的脚注允许用户创建对文档中提到的任何概念的引用。

通常，这些脚注位于文档的末尾，但是在Markdown中，您可以在文本中的任何位置定义它们。

创建脚注时，将以link的形式插入上标编号，将其插入到您创建的位置，当用户单击该idsihablep时，将直接将其带到包含注释参考的位置。

创建脚注的语法分为两部分。

第一部分代表包含link的数字，并使用方括号，尖号和标识符创建，例如: [^1].

标识符可以包含数字和字母，但不接受空格或制表符。

第二部分介绍了引用本身的内容，使用方括号，尖号后跟相同的标识符（您之前创建的标识符），冒号，空格和文本（示例）创建引用: [^1]: 我的脚注.

这是一个简单的脚注[^1]. 这是一个更详细的脚注[^bignote].

[^1]: 这是第一个脚注。

[^bignote]: 这是一个包含几个段落和代码的代码。

    缩进段落以将其包括在脚注中。

    `{ 我的密码 }`

    添加任意数量的段落。

HTML中的重编看起来像这样：

这是一个简单的脚注1. 这是一个更详细的脚注2.

  1. 这是第一个脚注。 ↩

  2. 这是一个包含几个段落和代码的代码。

    缩进段落以将其包括在脚注中。

    { 我的密码 }

    添加任意数量的段落。 ↩

注：页脚的内容可以放置在文档中的任何位置，但其他Markdown元素（例如表，列表和代码块）除外。
注：脚注将始终按顺序编号，无论您创建的标识符如何，音符序列[^ 35]，[^ yyyy]和[^ 102_444]都将呈现为1、2和3。

将ID或Class添加到Markdown元素

当我们在HTML中创建tags时，可以通过参数id和class为它们分配唯一标识和通用标识。

例子：

<h2 id="special-title"></h2> & <p class="bg-blue"></p>

使用id，我们可以获得特定元素的特定引用，通常该元素的id在HTML标记内是唯一的。

参数class是通用的，可以分配给几个元素。

id或class的功能是允许您通过JavaScript或CSS轻松识别这些元素，使用此id或此class我们可以处理属性例如颜色，大小，位置等

要将id或class添加到Markdown元素，请在该元素的内容之后打开花括号{，在其后加上一个井号＃表示id或"。指示class，然后是标识符本身，最后，关闭括号}。

例子：

### 我的头衔 {#identifier}

我的段落{.my-class}

HTML看起来像这样：

<h3 id="identifier">我的头衔</h3>

<p class="my-class">我的段落</p>
注意：请注意，并非所有Markdown应用程序都支持参数id或class的插入，因此始终重要的一点是查阅程序文档。
