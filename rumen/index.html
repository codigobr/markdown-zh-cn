什么是Markdown？

Markdown是一种用于格式化所有类型文档的标记语言，由John Gruber和Aaron Swartz于2004年创建，今天它已成为程序员中最受欢迎的语言之一。

根据维基百科：

注：标记语言是一种使用 tags 或符号对内容应用样式化的语言，世界上最受欢迎的标记语言是HTML，XML和Markdown。

在下面的图像中，您可以在文本编辑器Sublime Text中看到以Markdown格式格式化的文件。

Markdown或MD文件由纯文本组成，很容易理解，并且没有所有污染内容的HTML标记。

注：MD是Markdown的缩写。

使用Markdown，您可以使用符号标记将格式添加到文档中。

重要的是要注意Markdown格式与某些 WYSIWYG 编辑器使用的格式不同。

注意：WYSIWYG是 What You See Is What You Get 的缩写。基本上，内容的格式是实时发生的，即立即可见。要了解更多信息，请查阅维基百科文章。

例如，在WYSIWYG编辑器Microsoft Word中，通过选择特定内容并按下特定按钮，组合键或菜单选项来进行文本格式化。

在Markdown中，使用井号＃，星号*和下划线_等符号应用格式。

Markdown处理器使用此符号系统来了解将文件转换为HTML时应格式化的单词/短语。

例如，**我希望此文本为粗体**和_此文本以斜体_表示

太困惑了吗？让我们简化一下。

当您用两个星号**包围内容时，您将以粗体突出显示它，在第二个示例中，我们使用下划线_来应用斜体。

如何编辑Markdown文件

一个Markdown文件只不过是文本，因此任何简单的文本编辑器就足以完成任务。

但是，考虑到有趣的是：

  • 一个允许您将Markdown文档转换为另一种格式的编辑器。
  • 可以突出显示Markdown语法的编辑器。
  • 实时显示转换为HTML的编辑器

好消息是：

  • 有多种Markdown应用程序可用于各种操作系统，包括Windows，Linux，Android甚至iOS的Markdown应用程序。
  • 当涉及到Web时，情况变得更好，有几个站点提供免费的Markdown应用程序供您编辑文档online，最好的是，许多站点具有实时转换器，也就是说，在键入应用程序时，您将转换为HTML。
注意：值得记住的是Markdown格式非常简单，因此，使用该语言格式化的文档仍然可读。

Markdown和WYSIWYG

如果WYSIWYG编辑器（提供实时格式）非常方便，为什么我应该使用Markdown？

如果使用WYSIWYG编辑器，我可以按一个按钮并设置所需的格式，为什么我要输入带有Markdown标记的文档？

让我们看看使用Markdown的一些原因：

  • Markdown可用于创建几乎任何类型的文档，包括网站，书籍，演示文稿，e-mails，技术文档等等。
  • A portabilidade do Markdown é excepcional. Por ser um documento de texto simples, esse, pode ser editado em qualquer aplicativo de texto. Se contrastarmos com o Microsoft Word, o formato docx é proprietário e necessita de programa específico para ser aberto, o que não acontece com o md.
  • 可以在任何地方编辑"" MD文件。
  • 无论您使用哪种操作系统，Windows，Mac或Android，Markdown都是兼容的。
  • 即使发生灾难，也可以访问您的文档。您喜欢的文本编辑器不再起作用了吗？支持结束了吗？程序是否已付款？没问题，只需选择其他任何文本编辑器，然后继续编辑和访问您的文档，这对于诸如技术文档，书籍和TCC之类的文件而言极为重要。
  • 庞大的Markdown社区已使它可以在各种网站上用作格式帖子的标准语言进行访问。 GitHub和Reddit是使用Markdown的著名站点的示例。
  • 大量的人已经在使用Markdown，因此他们的格式化文档将被越来越多的人所理解。

如何转换Markdown文件

让我们了解Markdown应用程序将md文件转换为另一种格式（通常为HTML或PDF）的过程。

Markdown的简单性

您如何识别Markdown文件？通过扩展，对吗？通常是md或markdown。

Markdown文件是纯文本，没有格式，不是加密的文件，甚至更少，经过编译。

注意：此简单性允许程序分析您的文本中是否有识别Markdown元素的符号。

除编辑器外，几个Markdown应用程序还具有集成的转换器。转换器负责将Markdown语法转换为另一种格式。

转换如何完成

Markdown转换器使用Parser扫描其内容，以查找与Markdown兼容的标记。找到兼容标签时，Parser会检查输出格式是否匹配，例如，产生相同格式的tag HTML。

Markdown的使用

为Web创建内容是使用Markdown的最流行方法。不仅在HTML页面的生成中，而且在各种帖子的创建和编辑中，从创建简单的注释到详细的技术文档。

您还可以格式化任务列表，e-mails消息，购物清单等。

网站

与Markdown兼容的许多应用程序已经自动将您的内容转换为HTML。

您可以使用Markdown格式创建完整的网站。如果您要这样做，那么Jekyll和MkDocs之类的工具就可以满足您的需求。

Jekyll和MkDocs是从Markdown文档生成静态网站的工具。知道在创建网站时，有趣的是，您对HTML，CSS甚至使用Git的版本控制文件的熟悉程度降至最低。

GitHub页面为您的静态网站提供免费托管。

您使用WordPress吗？ WordPress.com上托管的站点已经支持Markdown，即使如此，您也可以将plugin Jetpack用于其他托管服务器上的站点。

格式化文字

Markdown的简单性免除了WYSIWYG编辑器的所有浮华选项，例如Microsoft Word，但它仍然提供了足够的格式来供您编辑文档。

该语言还支持粗体，斜体和删除线，此外还支持列表，标题，blockquotes等。

使用Markdown格式化文档的另一个优点是可以将它们转换为基本上任何其他文本文件格式。

以下是用于创建/编辑Markdown文档的应用程序列表：

  • Windows: ghostwriter 或者 Markdown Monster
  • Linux: ReText 或者 ghostwriter
  • Mac: MacDown, iA Writer 或者 Marked
  • Web: Dillinger 或者 StackEdit
  • iOS / Android: iA Writer

Markdown口味

想象一下以一种语言（语，发音，口音）存在的所有不同方言。即使对于讲母语的人，有时也很难理解一些东西。

就像语言一样，在不同程序中Markdown的实现也可能会有所不同，即使差异很小，也可能引起混乱。

Markdown的变体称为Flavors，例如，GitHub使用了更强大的Markdown版本GitHub-flavored Markdown或GFM。

一些应用程序仅使用基本语法，其他应用程序使用基本语法和扩展语法的组合，最后，还有一些应用程序偶发地实现了这两个元素。

该怎么办？

理解应用程序的最佳方法是查阅其文档。您也可以使用旧的反复试验策略。另请参阅我们的工具列表，其中列出了Markdown的哪些元素与每个应用程序兼容。

最终，最好的选择是使用提供最大兼容性的程序，例如Dillinger或StackEdit。

Markdown风味的一些示例是：GFM（GitHub-flavored Markdown）； MultiMarkdown； Markdown Extra，Kramdown。
