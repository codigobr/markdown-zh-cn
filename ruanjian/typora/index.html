Typora

Typora是一个简单的文本编辑器，对Markdown具有出色的支持。

概述

Typora是一个文本编辑器，它对Markdown的支持更加侧重于编辑新闻编辑室和报告。使用该工具构建包含多个文件的站点不是理想的。

该程序具有非常简单和干净的界面，但毫无疑问，即使看起来很简单，app仍具有多种配置。

该工具的一些有趣功能包括：使用键盘快捷键来插入Markdown格式；实时编辑器，可在用户键入时隐藏格式；直观易用的界面。

该工具中使用的Markdown的味道为GFM（GitHub-flavored Markdown）。

有关Typora的更多信息，请访问官方网站https://typora.io/。

Typora上的主题

使用Typora导出文档时，可以从该工具提供的多种主题中进行选择。

此外，您还可以选择使用CSS定制Typora主题。

要查看主题设置，请访问面板: Preferences > Appearance > Themes

Typora中的限制模式

Typora中的严格模式设置允许用户对Markdown格式施加某些限制。

例如，您可以指示在元素的构造中，有序列表应仅使用破折号-，而不使用星号*。

要对Typora施加限制，请访问面板: Preferences > Markdown > Syntax Preference

源代码模式

有时，该程序的实时编辑器可能会使更有经验的用户不满意，他们希望直接在文档中查看Markdown符号系统。

要激活源代码模式并显示Typora上的所有Markdown格式，请激活该选项: View > Source Code Mode.

导出选项

编辑Markdown文档结束时，我们可以选择按原样共享它，即与标记符号共享，或将其导出为另一种格式。

Typora提供了几个导出Markdown文件的选项。这些选项包括HTML，PDF，DOCX（Microsoft Word）甚至LaTeX。

要在Typora上导出Markdown文档，请导航至菜单File，然后单击选项Export。从Microsoft Word和LaTeX导出的DOCX之类的选项需要在计算机上安装Pandoc。
