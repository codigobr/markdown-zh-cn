Markdown Here

Markdown Here是Browser的扩展，可让您将网站上表单字段中键入的Markdown内容转换为Rich Text格式。

  1. Markdown
  3. Markdown出版商
  5. Markdown Here

概述

Markdown Here扩展使用户可以选择将Markdown带到任何地方。

至少可以说，该工具非常雄心勃勃，因为那里的不同站点数量都是天文数字，而且该应用程序也很冒险。

在所有？我的意思是，支持Rich Text格式的网站，是的，这种Microsoft Word格式，例如粗体，斜体和下划线。

诸如Gmail，Hotmail和WordPress之类的网站都是可以使用Markdown Here的地方的示例。

但是请不要停止，Markdown Here甚至可以在Evernote的Web版本上使用。

单击此处，您可以查询与该工具兼容的服务列表。

要使用Markdown Here，请先为您的浏览器安装扩展程序，然后访问Gmail之类的网站，使用Markdown格式键入email的内容，单击消息表单内mouse的相反按钮（右按钮），然后单击Markdown Toggle。

参见下图。

准备好了，Markdown Here将完成将站点本身的Markdown内容转换为Rich Text的工作。

当然，由于提供不同格式的网站数量众多，所呈现的内容可能会不一致。

在发送给他人之前，请务必检查邮件的格式，并确保其符合您的要求。

Markdown Here的官方网站位于https://markdown-here.com/。
